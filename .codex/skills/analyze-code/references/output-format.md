# Analyze Code Output Format

Use this exact structure:

## BUG HUNT SUMMARY
- Scope: files/modules reviewed
- Risk Level: Critical/High/Medium/Low

## CRITICAL FINDINGS
- Issue
- Impact
- Evidence (file:line)
- Fix

## POTENTIAL ISSUES
- Concern
- Risk
- Recommendation

## VERIFIED SAFE
- Checked area and result

## LOGIC TRACE
- Concise call/data path through relevant files

## RECOMMENDATIONS
1. Prioritized next actions
